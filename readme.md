# PyCharm 安装与配置指南

本指南提供了在 CentOS 7 虚拟机上安装 PyCharm 和配置 Python 环境的详细步骤，包括使用 Python 2.7 或 Python 3，安装必要的工具和设置镜像源等。

## 环境要求

- 操作系统：CentOS 7
- Python 版本：2.7（默认），推荐使用 Python 3
- PyCharm：社区版（Community Edition）
